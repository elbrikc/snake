# Project Title

Simple overview of use/purpose.

## Description

A simple stand-alone build of Snake for Waveshare 1.3-inch LCD header for RP Pico.

## Getting Started

### Dependencies

Requires Waveshare 1.3-inch LCD (https://thepihut.com/products/1-3-ips-lcd-display-module-for-raspberry-pi-pico-240x240) mounted to RP Pico.
I ran my Pico with 3 x AAA batteries and Schottky diode to allow for handheld system. But USB works fine too. 

### Installing

Upload RP Pico firmware to board using BOOTSEL button etc. Then upload repo contents to board. I used Thonny.

### Executing program

`main.py` should run automatically on standard boot.
